1. Git - это инструмент помогающий хранить данные на удаленке и при этом его разные версии.
2. База Данных нужна для хранения определенной информации в упорядоченном виде.
3. Primary Key - уникальный идентификатор каждой записи в таблице.
4. FOREIGN KEY - это ключ, используемый для связывания двух таблиц.
5. Строка хранит разные информации об одном определенном item, а столбец определенное инфо о разных itemaх.
6. CRUD - совокупность операции Create, Read, Update, Delete для работы с базами данных.
